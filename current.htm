arXiv swiper

Keeping up with the arXiv is at the core of research, but I sometimes find it hard to stay engaged, which is why I found myself building the "arXiv swiper".

This is a simple web app that I wrote on React (with a lot of help from Claude AI) that allows users to quickly browse through recent arXiv submissions in their field of interest by swiping left or right, similar to popular dating apps.

While I can't share the app itself because of arXiv policies, I have made the code available on GitHub for those interested in building their own version.

Find everything here: github.com/chiaralisotti/arxiv-swiper

Please, reach out if you have any questions or suggestions!

Possible improvements:

  • So far, I am loading the 20 most recent papers, and you can load 20 more once you're done reviewing the first 20. I would like to include a date picker.
  • User authentication would make this all a lot smoother, up to now you have to manually download your saved papers (or be careful to not close the active session!).
